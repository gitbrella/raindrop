Switch normal tasks from percentage based completion to checkboxes (complete vs not complete)

== Description ==

This plugin will allow you to change the percentage based tasks to checkboxes on a per phase basis.

== Installation ==

1. Upload 'panorama-checklists' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you have Project Panorama Professional installed and activated
4. Create or edit a project and create at least one phase and one task
5. Click the "Checklist" checkbox above the tasks
